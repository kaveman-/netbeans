Packaging Files Dialog Box

You can add files or directories to be included in the package.

To specify the contents of the package:

  1. For SVR4, IPS, RPM, or Debian packages, modify the package parameters on the Info tab as needed. See documentation for these package types for infomation about parameters.
  2. For all package types, add files to the package using the Files tab. In the Files tab, you specify the files to be packaged and map the path that each file should have in the package to the location of the file in your project. Type the root directory of the package in the Top Directory field. In the Files list, type the path to the file's location in the package in the File or Directory Path in Package column, and the location of the file in your project in the Original File or Link column.
    • Click New to add a blank line you can type into.
    • Click Add File or Directory to add a file or directory to the package. In the Add File or Directory dialog box, browse to the desired file or directory, or type a path name in the File Name field, and click Select.
    • Click Add Files From Directory to recursively add files from a selected directory. In the Add Files From Directory dialog box, browse to the desired directory, or type a path name in File Name field, and click Select.
    • Click Add Softlink to add a symbolic link to a file.
    • Click Duplicate to duplicate the selected line in the Files list.
    • Click Remove to delete the selected line in the Files list.
  3. Select the Default File Properties for regular files and executable files in the package. These properties will be used to set file access permissions when the package is installed.
  4. Click OK when your package information is complete.
