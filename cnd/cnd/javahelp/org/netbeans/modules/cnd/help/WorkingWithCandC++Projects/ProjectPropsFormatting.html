C/C++ Project Properties Dialog Box: Formatting

See Also 

The Formatting panel lets you specify formatting style used in the source editor for the selected project.

Use global formatting style. Select to use the global settings that are specified in the Tools > Options > Editor > Formatting properties.

Use project specific formatting style. Select to override the global formatting style in this project, and then select the style for each category.
