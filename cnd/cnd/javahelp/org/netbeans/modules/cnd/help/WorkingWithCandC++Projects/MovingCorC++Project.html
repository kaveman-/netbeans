Moving, Copying, or Renaming a C or C++ Project

When you create a project, the IDE bases much of its project metadata on the name and location of the project. These values are used in the project properties, and the names and locations of the project folders. In order to safely move, copy, and rename a project, you must use the commands provided by the IDE.

To move, copy, or rename a project:

  • Right-click the project's node in the Projects window and choose one of the following commands:
    • Rename Project
    • Move Project
    • Copy Project

Moving and renaming a project can break dependencies between projects. To fix the project dependencies, remove the renamed or moved projects from the required projects for any other projects, and then add them again.
