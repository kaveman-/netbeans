Creating and Modifying SendTo Scripts

See Also 

Each script has the following properties:

  • Script Name The name that identifies it in the SendTo menu
  • Script Type Determines whether the script acts on a selected file or selected text
  • Script. The command that runs the script when the name is selected, and the content of the script.

A script that acts on files only can also have the following:

  • Validation Script A script that is executed to determine whether the script is applicable to the selected files.

To display your existing scripts:

  1. Choose Tools > Options.
  2. In the Options dialog box, select Miscellaneous.
  3. In the Miscellaneous page, click the SendTo tab.

To modify an existing script:

  • In the SendTo tab, select the script from the Scripts List, change the name, type, or script, and click OK.

To add a new script, do one of the following:

  • In the SendTo tab, select an existing script from the Scripts List and click Copy. Change the name, type, or script of the copy, and click OK.
  • In the SendTo tab, click Add. Type the Script Name, select the Script Type, and type the Script. Click OK
